## About

Marketing website for **NextRide Handelsbolag**, a used car dealership based in Vargön, Sweden. Built as a solo project with a focus on clean design, brand consistency, and mobile responsiveness.

## Tech Stack

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS v3](https://tailwindcss.com/)
- Deployed on [Vercel](https://vercel.com/)

## Features

- Fully responsive – mobile first
- Brand-accurate design (gold & white, Cormorant Garamond serif)
- Smooth scroll navigation with mobile hamburger menu
- Real showroom and exterior images
- Google Maps embed for location
- Detailing price list
- Auto-highlights today's opening hours
- Direct links to Blocket car listings
- Clickable phone numbers on mobile
